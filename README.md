# 🏎️ F1 Telemetry Pit-Wall Dashboard

[![FastAPI](https://img.shields.io/badge/FastAPI-005571?style=for-the-badge&logo=fastapi)](https://fastapi.tiangolo.com/)
[![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white)](https://vitejs.dev/)
[![FastF1](https://img.shields.io/badge/FastF1-FF1801?style=for-the-badge&logo=formula1&logoColor=white)](https://github.com/theOehrly/FastF1)

A professional-grade, engineering-focused F1 telemetry visualization tool. Compare driver performances, analyze track maps, and monitor live-style standings directly from your browser.

Profesyonel düzeyde, mühendislik odaklı bir F1 telemetri görselleştirme aracı. Pilot performanslarını karşılaştırın, pist haritalarını analiz edin ve yarış klasmanını doğrudan tarayıcınızdan takip edin.

---

## 🔥 Features / Özellikler

- **🎯 Cockpit Layout:** Centered track map with isometric scaling for a perfect fit.
- **📊 Detailed Telemetry:** Scrollable data lane for Speed, Gear, Throttle, and RPM.
- **🌍 Dynamic Maps:** Real-time trajectory rendering with automated braking zone detection.
- **📱 Fully Responsive:** Optimized for everything from 3440px Ultrawide monitors to mobile phones.
- **🌐 i18n Support:** Full English (EN) and Turkish (TR) localization.
- **🏆 Live Standings:** Real-time race classification display.

---

## 🛠️ Tech Stack / Teknolojiler

- **Backend:** [FastAPI](https://fastapi.tiangolo.com/) & [FastF1](https://github.com/theOehrly/FastF1)
- **Frontend:** [Vite](https://vitejs.dev/) (Vanilla JS) & [Chart.js](https://www.chartjs.org/)
- **Styling:** Custom CSS with Glassmorphism effects.
- **Deployment:** Render (Backend), Vercel/Netlify (Frontend).

---

## 📦 Local Setup / Yerel Kurulum

### 1. Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```
